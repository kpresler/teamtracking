Welcome to the Software Engineering Team Tracking Application!

The purpose of this application is to support tracking student teams in software engineering courses (or any other courses that feature team-based projects) so that the teaching staff can observe and track team challenges, and offer support to teams in overcoming these challenges. This application is built around the Team Collaboration Reflection Survey (TCRS) from SIGCSE 2022
To start with, initialise the questions used in your version of the TCRS. Each question has a question text, which must exactly match against that in the CSV file you will submit later on. Additionally, questions have a type -- positive questions are ones where we would expect a successful team to answer "agree" or "strongly agree"; negative questions are ones where would expect successful teams to answer "disagree" or "strongly disagree". text questions are natural-language processed, and other questions are skipped when scoring the survey (but are still available for viewing later on).
Next, load in your TCRS submissions from a CSV file.
You can then view details for individual teams.
For a reference of the API and to view stored data, check out the API.
